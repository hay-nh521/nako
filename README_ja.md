# なにこれ
NakoはBashで書かれたAURヘルパーに生まれ変わった。名称の由来は音声ドラマ「桜木学園癒やし部」の登場人物である「赤平奈子」に由来する。

# 依存関係
* sudo
* pacman
* jq
* python3 (もしくはそれ以降の全ての版)
* curl
* pyalpm
* python-srcinfo
