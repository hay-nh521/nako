# Nako
-----
[日本語版はこちら](README_ja.md)

# What's this?
Nako has reborn as an AUR helper. The name comes from the character, Nako Akahira, in the audio drama, Sakuragi Academy Soothing Society.

# Dependencies
* sudo
* pacman
* jq
* python3 (or all later version)
* curl
* pyalpm
* python-srcinfo
